            ⚡ Encoding: longest phrase match (case‑sensitive). Unknown substrings stay as‑is.

            ⚡ Decoding: unknown tokens become  token  (spaced) to preserve separation.

            ⚡ Multi‑line: each line is processed independently; output preserves line breaks.
